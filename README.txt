Aplición web que simula un pequeño sysacad de una universida.
Los usuarios alumnos pueden ver e inscribirse a las materias que desean cursar.Los usuarios administradores pueden gestionar los prefesores y las materias a ofrecer.
El lenguaje de programación principal utilizado de la aplicación es C# con el framework ASP.NET.  La base de datos que utilicé es Microsoft SQL Server 2019, haciendo las conexiones en el back mediante ADO.NET Entity Framework.
Para las consultas utilizo queries, views, stored procedures con y sin parámetros y transacciones.
Para el front utilicé el framework de CSS, Bootstrap 4.
Para las validaciones utilicé JavaScript.
